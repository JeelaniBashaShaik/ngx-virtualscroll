Inputs

templateId

Ideally, a template is defined between the 'ng-template' tags and a name is given to the template starting with '#', followed by the template name. This name is called as the template reference variable. This reference variable is passed to the 'templateId' property.

In the above image, 'mywish1' is the template reference variable

rows

This property accepts an array of objects/numbers/strings which are to be iterated in the virtual-scroll component

height

This property accepts a number which will be used to set the height of virtual-scroll component in pixels

width

This property accepts a number which will be used to set the width of virtual-scroll component in pixels

itemHeight

This property accepts a number which will be used to set the height of each template in the virtual scroll and is measured in pixels
