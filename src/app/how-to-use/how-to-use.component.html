How to Use

Step 1

npm install -s @jb7/ngx-virtualscroll

Step 2

Add NgxVirtualScroll module to ur app.module.ts

Step 3

Define a template with reference variable. assign a dummy variable named linkObj to the template

Dont forget to assign the linkObj (let-linkObj = ) property template. Also have an empty object named 'linkObj' in your .ts file. If array of objects have to be shown in virtual scroll, pass the properties of object as shown in image

Step 4

Put out the 'ngx-virtualscroll' selector with giving rows, height, itemHeight and the templateId properties

Hola !!!!!

You are up and running with virtual scroll
